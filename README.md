# Advanced CLI calculator

A advanced CLI calculator which support basic operation, advanced operation, financial operation. In this app i have not used inbuild math module. Complex algorithms implemented from scratch.

## Operations supported

- Basic operatios: Addition, Subtraction, Multiplication, Division.
- Advanced operations: Square root, Factorial, Exponents, nth root, Logarithms, Sine function.
- Financial operations: EMI calculator, Pencentage calculator.
